# Asyncio practice

pub/sub example

참고: [asyncio: We Did It Wrong](https://www.roguelynn.com/words/asyncio-we-did-it-wrong/)

## Required

Python 3.7+

## Run

### pipenv

```bash
pipenv install --dev
pipenv shell && python -m app
```

### virtualenv

```bash
virtualenv .venv
source .venv/bin/activate
pip install -r requirements.txt
python -m app
```
